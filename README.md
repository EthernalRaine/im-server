## Features

MySpaceIM (MSIM Protocol):
* MSIMv1 - MSIMv7 Supported
* Authentication (RC4)
* Instant Messages
* Status Messages
* Offline Messages
* "Zaps" (Action Messages)
* Typing Indicators

AIM and ICQ (OSCAR Protocol):
* Authentication (FLAP, BUCP)
